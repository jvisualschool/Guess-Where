# 🌍 GUESS WHERE - 전세계 랜드마크 퀴즈 게임

![GUESS WHERE Splash](/splash.jpg)

**전세계의 유명한 장소들을 사진의 일부만 보고 맞춰보세요!** 
현대적이고 세련된 디자인과 함께 즐기는 인터랙티브 퀴즈 게임입니다.

---

## 🚀 주요 기능 (Features)

-   **인터랙티브 퀴즈:** 사진의 아주 작은 조각부터 시작하여 힌트를 통해 점차 넓은 영역을 확인하며 정답을 추측합니다.
-   **5단계 힌트 시스템:** 단계별로 확장되는 원형 마스크 효과를 통해 몰입감 있는 게임 경험을 제공합니다.
-   **무작위성:** 매 판마다 퀴즈 순서가 섞이며, 힌트가 나타나는 위치를 랜덤하게 설정할 수 있어 매번 새로운 재미를 줍니다.
-   **고품질 콘텐츠:** 50개 이상의 전세계 유명 랜드마크 데이터(이름, 국가, 설명, 구글 지도 링크)를 포함하고 있습니다.
-   **시청각 효과:** 게임 시작과 정답 공개 시 즐거운 사운드 효과와 화려한 컨페티(꽃가루) 효과가 분위기를 고조시킵니다.
-   **관리자 모드:** 자물쇠 아이콘을 통해 진입할 수 있는 대시보드에서 이미지 URL을 실시간으로 수정하고 관리할 수 있습니다.

## 🛠 기술 스택 (Tech Stack)

-   **Frontend:** [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)
-   **Build Tool:** [Vite](https://vitejs.dev/)
-   **Animation:** [Framer Motion](https://www.framer.com/motion/)
-   **Icons:** [Lucide React](https://lucide.dev/)
-   **Effects:** [Canvas Confetti](https://www.npmjs.com/package/canvas-confetti)
-   **Styling:** Vanilla CSS (Modern Premium Look)

## 📋 설치 및 실행 방법 (Installation)

1.  **레포지토리 클론:**
    ```bash
    git clone https://github.com/your-username/guess-where.git
    cd guess-where
    ```

2.  **의존성 설치:**
    ```bash
    npm install
    ```

3.  **로컬 개발 서버 실행:**
    ```bash
    npm run dev
    ```

4.  **빌드 및 배포:**
    ```bash
    npm run build
    ```

## 🔐 관리자 모드 (Admin Access)

-   메인 화면 우측 상단의 **자물쇠 아이콘**을 클릭합니다.
-   비밀번호(`071300`)를 입력하여 접속합니다.
-   모든 퀴즈의 이미지 URL을 직접 수정하고 저장할 수 있습니다.
